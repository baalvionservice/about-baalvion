# **App Name**: Baalvion Nexus

## Core Features:

- Dynamic Content Management (CMS): An intuitive admin panel allowing authorized users to create, edit, and delete all website content dynamically (e.g., pages, sections, projects, ecosystem details) through an API-driven interface.
- API-Driven Public Website: A modern Next.js front-end that fetches and renders all public-facing content (Homepage, Company, Platform, Ecosystem, Projects, etc.) dynamically from a 'Firestore-like' mock backend via robust REST APIs.
- Project Showcase & Management: A dedicated 'Projects' page displaying detailed information for various projects categorized by type (e.g., Core Platform, Industrial, Intelligence), including their Name, Description, and Status, managed through the CMS.
- Inquiry Submission & Tracking: A user-friendly 'Contact' form for visitors to submit inquiries, with an integrated admin view within the CMS for efficient management and review of all submissions.
- Secure Admin Authentication: A basic login system protecting the /admin dashboard, ensuring that only authorized personnel can access and manage the website's content.
- AI Content Drafting Tool: An integrated AI tool within the CMS to assist administrators in drafting initial content for new pages, sections, or project descriptions based on keywords and existing corporate guidelines, enhancing content creation efficiency.
- Responsive Card-Based Interface: A clean, modern, and fully responsive user interface utilizing sophisticated card-based layouts for consistent presentation of information across all devices, from mobile to large desktop displays.

## Style Guidelines:

- Color scheme: Dark premium theme for professionalism and sophistication. Primary actions and accents will use a sophisticated mid-tone blue (#2760B8). The background will be a deeply desaturated, dark blue-grey (#151B24). An accent color of bright cyan (#5CD1E5) will highlight interactive elements and important calls to action.
- Font: 'Inter' (sans-serif) will be used across the entire site for its clean, modern, and highly legible characteristics, suitable for both headlines and body text to convey clarity and professionalism.
- Use a set of modern, minimalist line icons that align with enterprise technology and global connectivity. Icons should maintain a professional and subtle presence, enhancing readability without cluttering the interface.
- Content will be structured using card-based layouts, providing a clean, organized, and easily digestible presentation of information. Emphasis on clear spatial relationships between elements and modular components to enhance maintainability and user experience. Full responsiveness across all screen sizes is paramount.
- Implement subtle, elegant animations and transitions for navigation, interactive elements, and content loading. Focus on smooth, deliberate movements that enhance the premium feel and guide the user's attention without being distracting or excessive.
